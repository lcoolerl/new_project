# Инструкция по языку MarkDown
## Форматирование текста

Новая строка через 2 Enter.
Иначе продолжит в этой же строке

Обычный текст.

*курсив*

**полужирный**

## Списки
(переход на новую строку с спецсимвола)
### Обычный
* список 1
* список 2
* список 3

1. Список 1
2. Список 2
